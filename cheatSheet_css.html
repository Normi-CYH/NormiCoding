CSS, or Cascading Style Sheets, is a style sheet language used to describe the presentation of a document written in HTML. It controls the layout, formatting, and overall appearance of a webpage, allowing developers to define styles for various elements.

Selectors
p {...}
Name selector, name is the elements in HTML.
#myID {...}
ID selector, ID is the id attribute of elements in HTML.
.myClass {...}
Class selector, class is the class attribute of elements in HTML.
* {...}
Universal selector, mean all the elements in HTML.
p, div, span {...}
Group selector, mean some of the elements in HTML.
div a {...}
Descendant selector, the descendant selector matches all elements that are descendants of a specified element.
div > a {...}
Child selector, the child selector selects all elements that are the children of a specified element.
div + a {...}
Adjacent sibling selector, the adjacent sibling selector id used to select an element that is directly after another specific element.
div ~ a {...}
General sibling selector, the general sibling selector selects all elements that are next siblings of a specified element.
p::after {...} |
::before, ::first-letter, ::first-line, ::marker, ::selection
Pseudo element selectors, a CSS pseudo-element id used to style specified parts of an element.
a:hover {...} |
:active, :focus, :link, :visited, :checked, :disabled, :first-of-type, :last-of-type, :not(selector), :nth-of-type(n), :nth-of-type(n)
Pseudo class selectors, a pseudo-class is used to define a special state of an element.
p[attribute] {...} |
[attribute=value], [attribute~=valur], [attribute|=value], [attribute^=valur], [attribute$=value], [attribute*=value]
Attribute selectors, the attribute selectors is used to select elements with a specified attribute.
Background
background: bg-color bg-image position bg-size bg-repeat bg-origin bg-clip bg-attachment;
Sets all the background properties in one declaration.
background-color: color/ transparent;
Sets the background color of an element.
background-image: URL/ none;
Sets the background image for an element.
background-repeat: repeat/ repeat-x/ repeat-y/ no-repeat;
Sets how a background image will be repeated. By default, the background-image property repeats an image both horizontally and vertically.
background-position: left top/ left center/ left bottom/ right top/ right center/ right bottom/ center top/ center center/ center bottom/ x%y%/ xpos ypos;
Sets the starting position of a background image.
background-attachment: scroll/ fixed/ local;
Sets whether a background image is fixed or scrolls with the rest of the page.
background-size: auto/ length/ cover/ contain;
Specifies the size of the background image.
background-clip: border-box/ padding-box/ content-box;
Specifies the painting area of the background.
Borders
border: border-width border-style border-color;
Sets all the border properties in one declaration.
border-width: medium/ thick/ thin/ length;
Sets the width of the four borders.
border-style: dotted/ dashed/ solid/ double/ groove/ ridge/ inset/ outset/ hidden/ none;
Sets the style of the four borders.
border-color: color/ transparent;
Sets the color of the four borders.
border-radius: length/ %;
Sets all the four border radius properties for rounded cornres.
border-top/ border-left/ border-bottom/ border-right
All the border properties can add the direction to specifies the style of differe direction borders.
Margins
margin: length/ auto;
This property is used to create space around elements, outside of any defined borders.
margin-top/ margin-left/ margin-bottom/ margin-right
The margin property can add the direction to specifies the width of differe direction margins.
Paddings
padding: length;
This property is used to create space around element's content, inside of any defined borders.
padding-top/ padding-left/ padding-bottom/ padding-right
The padding property can add the direction to specifies the width of differe direction padding.

float: none/ left/ right;
This property is used for positioning and formatting content.

Specifies the stack order of an element.
